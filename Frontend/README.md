## Local development

Requirements: Node.js and `npm` (or `yarn`) as the package manager.

Quickstart:

```sh
# Clone
git clone <YOUR_GIT_URL>
cd <YOUR_REPO_DIR>

# Install dependencies (npm / yarn)
npm install
# or
yarn install

# Run dev server
npm run dev
```
